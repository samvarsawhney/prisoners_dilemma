# Prisoner's Dilemma
The Prisoner's Dilemma is a fundamental game theory concept illustrating why two rational individuals might not cooperate, even if it appears in their best interest to do so. 

# How to run
Run following command to run 10000 rounds of Prisoner's Dilemma game. At end it will display total scores of different players. The name of each player reflects the strategy used by that player.

python3 Main.py
